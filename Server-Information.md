# 🎮 Multiplayer Information - `vAlpha 2.0`
> Last updated: March 30, 11:02 AM

---

## 🛰️ Joining via Essential
We are using the **Essential Mod** for multiplayer. This means there is no dedicated server IP. 

To join, ensure you have the modpack installed and have added the host as a friend within the Essential interface.

- **Mod Loader:** Forge `v47.4.10`
- **Minecraft Version:** `1.20.1`

---

## 🚦 Operational Constraints
Performance is dependent on the host's local machine. Please adhere to the following guidelines to maintain stability:

*   **👥 Player Cap:** Recommended **6-8 players** max.
*   **🌍 Chunk Loading:** A maximum of **2 players** should be generating new chunks (exploring new territory) simultaneously.
*   **👾 Entity Limits:** Local entity counts should not exceed **200-300** in a single concentrated area.
*   **💎 Optimization:** This pack is optimized for stability, but intensive machines should be kept to a reasonable limit.
